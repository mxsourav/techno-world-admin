# 🔧 Techno World Books — Admin Panel

Standalone React admin panel for the Techno World Books e-commerce platform.

## Features
- **Dashboard** — Real-time stats (Revenue, Orders, Users, Books)
- **Product Management** — Full CRUD + Stock Control
- **Order Management** — Order status tracking
- **Bulk CSV Import** — Analyze → Preview → Execute workflow
- **Media Manager** — Upload, preview, delete book covers & PDFs
- **Homepage CMS** — Edit hero, flash sale, featured books, testimonials, sale banner
- **Book Preview** — Per-book cover & PDF upload modal

## Tech Stack
- React 19 + TypeScript + Vite
- Tailwind CSS 4
- Lucide React Icons

## Getting Started

```bash
git clone https://github.com/mxsourav/techno-world-admin.git
cd techno-world-admin
npm install
npm run dev
```

**Note:** This panel requires the `techno-world-api` backend running.

## Deploy to Render
Includes a `render.yaml` for automatic deployment to Render.com as a static site.
